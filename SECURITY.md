# Security Policy

## Segnalare una vulnerabilità

Se hai trovato una vulnerabilità di sicurezza, **non aprire una issue pubblica**.

Invia una email a: info@gbdesign.it
Oggetto: [SECURITY] descrizione breve del problema

Risponderemo entro 48 ore lavorative. La vulnerabilità verrà corretta prima di qualsiasi disclosure pubblica.
